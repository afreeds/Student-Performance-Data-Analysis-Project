# 📊 Student Performance Analyzer

> **A Python-based data analysis project that transforms raw student marks into meaningful insights through statistics, grading, and visual analytics.**

---

## 🚀 About the Project

**Student Performance Analyzer** is a beginner-friendly data analysis project developed using **Python**. It demonstrates how educational datasets can be processed to generate useful academic insights.

Instead of manually evaluating marks, this application automatically calculates each student's average score, assigns grades based on performance, and presents the results using colorful visualizations.

Whether you're learning **Python**, **Pandas**, or **Matplotlib**, this project serves as a practical example of real-world data analysis.

---

## ✨ Highlights

* 📂 Import student records from a CSV dataset
* 📈 Analyze academic performance automatically
* 🧮 Calculate average marks for every student
* 🎯 Assign grades using predefined criteria
* 📊 Display descriptive statistical analysis
* 📉 Generate professional charts for better understanding
* 🐍 Beginner-friendly Python implementation

## ⚙️ Installation

Clone the repository

```bash
git clone <repository-url>
```

Navigate to the project

```bash
cd Student-Performance-Analyzer
```

(Optional) Create a virtual environment

```bash
python -m venv venv
```

Activate the environment (Windows)

```bash
venv\Scripts\activate
```

Install dependencies

```bash
pip install -r requirements.txt
```

---

## ▶️ Run the Application

```bash
python student_analysis.py
```

## 🎓 Grading System

| Average Score | Grade |
| ------------- | ----- |
| 90 - 100      | ⭐ A   |
| 75 - 89       | ✅ B   |
| 60 - 74       | ✔ C   |
| Below 60      | ⚠ D   |
